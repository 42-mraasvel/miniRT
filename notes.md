Colors:

	Integer format: 0xTTRRGGBB
		T: Transparency
		R: Red color
		G: Green Color
		B: Blue color

Initialization:

	1. include <mlx.h> header to access all the functions
	2. <mlx_init> function returns a void* to current MLX instance
